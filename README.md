# Alim Archetype AI

Faceless relationship intelligence — screenshot in, brutal clarity out.

## Run locally

```bash
cd alim-archetype-ai
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) (EN) or [http://localhost:3000/tr](http://localhost:3000/tr) (TR).

## Stack

- **Next.js 15** + TypeScript + Tailwind CSS v4
- Deploy: **Vercel**
- AI (next): OpenAI / Anthropic vision API
- Payments (next): Stripe (+ Iyzico for TR)
- DB (next): Supabase
